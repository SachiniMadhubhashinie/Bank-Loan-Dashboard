# Bank-Loan-Dashboard
Developed an interactive loan analytics dashboard to track key KPIs such as total loan applications, funded amount, amount received, interest rate, and DTI. The project includes summary, overview, and detailed dashboards to analyze loan performance, borrower behavior, and regional trends for data-driven decision-making.

<h2>Project overview </h2>
<p>This repository contains a Bank Loan Analysis project built using SQL. The project focuses on analyzing loan applications, funded amounts, repayments, interest rates, and borrower financial health. It helps understand loan portfolio performance through KPIs, trends, and status-based insights.<br>

The project is ideal for beginners and intermediate learners to practice aggregation, filtering, grouping, date functions, and business analytics using SQL.</p>
